Jest setup for tests:
	
	npm i jest --save-dev

	npm i jest-cli -g

	npm i jest-environment-jsdom --save-dev


Run tests: 

	jest
	
	jest --coverage

Install canvas:
	npm install --save-dev canvas
	

Lint:
	npm install eslint -DE
	// npm init @eslint/config 
	
To run lint:
	npm run lint


JS Doc
	npm install -g jsdoc
	
	 jsdoc index.js
